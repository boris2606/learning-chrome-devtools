JavaScript - Налагодження та Дебаг (Chrome DevTools)

Методи для прошуку помилок

Для дебагу краще вього використовувати браузер Google Chrome. Використовуючи наступні методи:

  • Використовуючи console.log()
  • Використовуючи console.table(змінна) буде змога переглянути у вигляді таблиці що знаходиться в масиві/об'єкті у зручному форматі
  • Використовуючи вкладку console використовуючи інструмент розробника в браузері
  • Використовуючи вкладку source використовуючи інструмент розробника в браузері. Є змога працювати в breakpoints. Обравши в JS файлі рядок на якому потрібно поставити breakpoint. В результаті чого коли дойдемо до даного скрипту, буде виконано зупинку і показано результат на даний момент
  • Використовуючи слово debugger в коді JS, автоматично на ньому буде спрацьовувати breakpoint
  • Використовуючи console.trace() є змога відслідкувати які функції були викликані та виконані
  • Використовуючи:
    console.time('timer')
    // тут знаходиться той код який потрібно перевірити
    console.timeEnd('timer')
     Присутня змога переглянути на швидкість виконання даного коду
  • Використовуючи вкладку network в браузері , для перевірки мережевих взаємодій
